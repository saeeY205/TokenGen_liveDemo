# TokenGen Live Demo

A standalone, web-based demonstration of the TokenGen Queue Management system.
This is Live Demo of my application Token Generator for Queue Management.

## 🛠️ Features
- **Standalone Mobile UI**: View the app inside a phone-style frame on any browser.
- **Mock Firestore**: Fully functional queue logic without the need for a Firebase backend.
- **In-Memory State**: Real-time updates for "Join Queue" and "Call Next" simulations.
- **Admin & Student Roles**:
  - Admin: `admin@gmail.com` / `12345`
  - Student: Create your own demo account or use any sign-in.

## 📦 Tech Stack
- **Framework**: Flutter Web
- **Backend Simulation**: Mock Services (Auth, Firestore)
- **CI/CD**: GitHub Actions
- **Hosting**: GitHub Pages

---
*Note: This is a frontend-only demo. Refreshing the page will reset the queue state.*
